IMAGES FOLDER - WebJysk Marketplace
====================================

Place your images here. The app will use them as follows:

- hero-banner.jpg     = Home page hero background (already added)
- categories/         = Optional: category-1.jpg, category-2.jpg, etc. for category cards
- products/           = Product images come from the API (admin uploads)

FREE IMAGE SOURCES:
- Pixabay: https://pixabay.com (free, no attribution)
- Unsplash: https://unsplash.com (free, high quality)
- Freepik: https://www.freepik.com (free with attribution)
- Pexels: https://www.pexels.com (free)

For furniture/home decor: search "furniture", "living room", "Scandinavian"
